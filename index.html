Comment jouez au test ?

1. Choisissez la langue et le niveau dans le slider à votre gauche puis lancer le test.

2. Vous arriverez sur le test en question puis lisez la question posée.

3. Vous pourrez former votre question en cliquant sur les mots du bas et s'afficheront dans l'ordre dans la bordure

4. Pour vérifier votre réponse, cliquer sur le bouton pour vérifier votre réponse.

Vous souhaitez voir à quoi ressemble le test ? Cliquez sur le lien pour accéder à une démo dans la langue choisie Accéder à la démo
